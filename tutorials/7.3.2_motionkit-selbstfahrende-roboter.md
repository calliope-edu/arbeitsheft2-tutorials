# 7.3.2 MotionKit: Selbstfahrende Roboter

## Aufgabe @showdialog
Eine Aufgabe des autonomen Fahrens ist es, Hindernisse zu erkennen und diesen auszuweichen. Mithilfe des Ultraschallsensors lassen sich Hindernisse erkennen und umfahren.


## Aufgabe 1

test

```blocks
basic.forever(function () {
    while (maqueen.ultrasonic(maqueen.DistanceUnit.Centimeters) > 10) {
        maqueen.motorRun(maqueen.Motors.All, maqueen.Dir.CW, 200)
    }
    maqueen.motorStop(maqueen.Motors.All)
})
```

## Aufgabe 2

test

```blocks
basic.forever(function () {
    while (maqueen.ultrasonic(maqueen.DistanceUnit.Centimeters) > 10) {
        maqueen.motorRun(maqueen.Motors.All, maqueen.Dir.CW, 200)
    }
    maqueen.motorStop(maqueen.Motors.All)
    maqueen.motorRun(maqueen.Motors.All, maqueen.Dir.CCW, 50)
    basic.pause(500)
    maqueen.motorRun(maqueen.Motors.M1, maqueen.Dir.CCW, 50)
    maqueen.motorRun(maqueen.Motors.M2, maqueen.Dir.CW, 50)
    basic.pause(500)
})
```
